### NoteApp-Front

The frontend for a note management app.

## Features

- Authentication (basic, OAuth i.e. Google)
- Nested folder structure with notes inside each one for organization
- VS code like layout i.e. explorer on the left and text editor on the left
- Rich text editor with text formatting, images (drag and drop, and paste) and code blocks etc
- Auto save

## Features To Come

- Note Share Link

## Tech Stack

- Next.js, Tailwind, Shadcn, Lucide React

## Run Locally

You can't run it locally because 90% of the app is depending on the backend
if there is no backend only login, signup and home page are available.
